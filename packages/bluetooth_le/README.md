# bluetooth_le (planned)

Bluetooth Low Energy (GATT) for Dart and Flutter — sibling to
[`bluetooth_classic`](../bluetooth_classic). **Not yet implemented.** See
[DESIGN.md](DESIGN.md) for the intended architecture.

If you need to reach a **non-MFi device from iOS**, this is the package you'll
want — ExternalAccessory (Bluetooth Classic) only works with MFi accessories.
